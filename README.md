## Download and Launch Auzia

```sh
git clone https://www.github.com/SZinedine/auzia-conky
cd auzia-conky
conky -c conkyrc
```

Edit `settings.lua` to choose your network interface, Internet speed and other settings.

***

## Dependencies

- Conky 1.10+
- cairo
- imlib2
